# Character puppet assets

Drop `assets/character/` into your app public/assets or src/assets folder. Use `characterRigConfig.json` as the starting pivot/offset config.

These are placeholder separated transparent PNGs designed for a PixiJS puppet rig. Replace each PNG with your final art using the same filenames.

Important pivots:
- upper arms pivot at shoulders
- lower arms pivot at elbows
- hands pivot at wrists
- upper legs pivot at hips
- lower legs pivot at knees
- feet pivot at ankles
- head pivots near neck
